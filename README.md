This work was inspired by the Reward Machines framework introduced by Toro Icarte et al. (2022), which shows how exposing reward function structure can improve sample efficiency in reinforcement learning.

I wanted to try recreate the environment and the techniques mentioned in the paper, which I found to be very interesting. It is very pratical and provide considerable improvement to sample efficiency.

Over multiple trials I was able to recreate how CRM is able to converge faster than the cross product baseline in the multitask officeworld senario proposed in the paper.
